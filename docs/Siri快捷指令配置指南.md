# Siri 快捷指令配置指南

> 配置完成后，你只需要对 Siri 说「今晚不回家」或「谁回家」就能操作。

## 前置准备

1. 打开 https://huijibu.onrender.com ，注册并登录
2. 进入 **设置** → **API Token**，点击「显示」再点击「复制 Token」
3. 记下你的 Token，后面配置要用

---

## 快捷指令一：今晚不回家

对 Siri 说 **「今晚不回家」** → 自动将状态改为不回家。

### 配置步骤

1. 打开 iPhone 上的 **快捷指令** App
2. 点击右上角 **+** 创建新快捷指令
3. 点击顶部名称，改名为 **「今晚不回家」**
4. 依次添加以下操作：

#### 步骤 1：添加「要求输入」（可选，用于填写原因）

- 搜索添加 **「要求输入」** 操作
- 问题设置为：`不回家的原因？`
- 输入类型选择 **文本**
- 默认回答填：`加班`

#### 步骤 2：添加「获取 URL 内容」

- 搜索添加 **「获取 URL 内容」** 操作
- 配置如下：

| 字段 | 值 |
|------|-----|
| **URL** | `https://huijibu.onrender.com/api/v1/status/me` |
| **方法** | `PUT` |
| **请求体** | `JSON` |

- 点击 **「头部」**，添加一条：

| 键 | 值 |
|----|-----|
| `Authorization` | `Bearer 你的API_TOKEN`（替换成你复制的 Token） |

- 点击 **「请求体」**，添加两个字段：

| 键 | 类型 | 值 |
|----|------|-----|
| `going_home` | 布尔值 | `false` |
| `reason` | 文本 | 选择上一步的「提供的输入」变量 |

#### 步骤 3：添加「显示通知」

- 搜索添加 **「显示通知」** 操作
- 内容设置为：`✅ 已标记今晚不回家`

#### 完成

点击右上角 **「完成」** 保存。

现在对 Siri 说 **「嘿 Siri，今晚不回家」** 试试！

## 快捷指令二：今晚回家

对 Siri 说 **「今晚回家」** → 将状态改回回家。

### 配置步骤

1. 创建新快捷指令，命名为 **「今晚回家」**
2. 添加 **「获取 URL 内容」** 操作：

| 字段 | 值 |
|------|-----|
| **URL** | `https://huijibu.onrender.com/api/v1/status/me` |
| **方法** | `PUT` |
| **请求体** | `JSON` |

- 头部添加：

| 键 | 值 |
|----|-----|
| `Authorization` | `Bearer 你的API_TOKEN` |

- 请求体添加：

| 键 | 类型 | 值 |
|----|------|-----|
| `going_home` | 布尔值 | `true` |

3. 添加 **「显示通知」**：`✅ 已标记今晚回家`
4. 保存

---

## 快捷指令三：谁回家

对 Siri 说 **「谁回家」** → 查看今晚所有家人的状态。

### 配置步骤

1. 创建新快捷指令，命名为 **「谁回家」**

2. 添加 **「获取 URL 内容」** 操作：

| 字段 | 值 |
|------|-----|
| **URL** | `https://huijibu.onrender.com/api/v1/status` |
| **方法** | `GET` |

- 头部添加：

| 键 | 值 |
|----|-----|
| `Authorization` | `Bearer 你的API_TOKEN` |

3. 添加 **「获取词典值」**：
   - 获取 `URL 内容` 的第 1 个项目

4. 添加 **「获取词典值」**：
   - 键：`summary`

5. 添加 **「朗读文本」** 或 **「显示结果」**：
   - 内容选择上一步获取到的 `summary` 值

6. 保存

现在对 Siri 说 **「嘿 Siri，谁回家」**，Siri 会告诉你「今晚 3 人回家，1 人不回家」。

## 进阶：查看每个人的详细状态

如果你想让 Siri 逐个报出谁回家、谁不回家：

1. 在「谁回家」快捷指令中，获取到 URL 内容后
2. 添加 **「获取词典值」**：键填 `members`（第一个元素）
3. 添加 **「对每个项目重复」** 循环
4. 在循环内：
   - **获取词典值**：键 `nickname` → 存为变量 `名字`
   - **获取词典值**：键 `going_home` → 存为变量 `回家`
   - **如果** `回家` **等于** `true`
     - 文本：`「名字」回家`
   - **否则**
     - 获取词典值：键 `reason`
     - 文本：`「名字」不回家，原因：reason`
5. 循环结束后，**合并文本**（以换行分隔）
6. **朗读文本**

Siri 会说：「老齐回家，小齐不回家，原因：加班」。

## 小贴士

- **快捷指令名就是 Siri 唤醒词**：指令命名为「今晚不回家」，就可以说「嘿 Siri，今晚不回家」
- **添加到主屏幕**：在快捷指令编辑页 → 点击顶部名称 → 「添加到主屏幕」，可以一键触发
- **Apple Watch**：快捷指令会自动同步到 Apple Watch，抬手就能操作
- **自动化**：在快捷指令 App 的「自动化」标签页，你还可以设置定时提醒（如每天下午 5 点提醒更新状态）
- **Token 安全**：API Token 等同于你的密码，不要分享给他人

## 常见问题

**Q: 第一次调用很慢？**  
A: Render 免费服务在 15 分钟无访问后会休眠，首次唤醒约需 30-60 秒。之后就快了。

**Q: Token 在哪里获取？**  
A: 打开 https://huijibu.onrender.com → 登录 → 设置 → API Token → 显示 → 复制

**Q: 可以在 iPad / Mac 上用吗？**  
A: 可以，快捷指令在 Apple 全设备同步。Mac 上也可以说「嘿 Siri，今晚不回家」。

**Q: 加入了多个家庭怎么办？**  
A: 在 URL 后面加上 `?family_id=你的家庭ID`，家庭 ID 可以在设置页面查看。
